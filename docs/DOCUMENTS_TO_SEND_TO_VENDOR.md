# Documents to Send to Developer (Vendor)  
## Contract Between Website Owner and Developer

Use this checklist when sending the contract and supporting documents to the **Developer** (vendor).

---

## What you are sending

This is the **contract between the Website Owner** (you, the client) **and the Developer** (the vendor building the site). The main document defines both parties’ obligations, review process, and billing.

---

## Package contents

- [ ] **VENDOR_DELIVERABLES_AND_SCOPE.md** – Contract covering:
  - **Parties:** Website Owner (client) and Developer (vendor)
  - **3× website draft reviews** (Owner gives feedback; Developer implements)
  - **3× logo draft reviews** (Owner gives feedback; Developer implements)
  - **Facebook & Instagram ads** (Developer delivers as agreed)
  - **Brief website workflow**
  - **Billing:** AUD 2,200 (Owner pays Developer)
  - **Further updates** may cost extra; to be quoted separately
- [ ] **This checklist** (optional)
